Hue + Kotlin Mutliplatform

Shade is an SDK for communicating with Hue lights with modern Kotlin data structures and Coroutines.

Github Latest Release

SDK

Shade's SDK uses straightforward suspending functions to control lights and groups, making it easy to get started: 

suspend fun turnOnLights() {
    shade.groups.setState(
        GROUP_ALL,
        GroupStateModification(
            on = on,
            brightness = 50.percent,
            colorTemperature = 5000.kelvin,
        )
    )
}

// Replace `+` with latest version, if desired.
implementation("com.inkapplications.shade:core:+")
